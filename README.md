# DBAdmin
## 一个数据库管理后台

### 1、包括以下功能：
```
1、用户管理
2、单个用户的数据库管理
3、单个用户的表查看
4、单个用户的数据查看
```

### 2、启动
```
pip install -r requirements.txt

python3 run.py
```
地址：http://127.0.0.1:5000/

### 3、登录
```
账号：root
密码：123456
```

### 4、数据库配置方式
```
database_name:  demo
database_ur:    mysql://root:123456@127.0.0.1:3306/demo
```
